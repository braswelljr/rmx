# rmx

A cross-platform replacement for the UNIX `rm` command.

## Installation

```bash
go get github.com/braswelljr/rmx
```

## Usage

```bash
rmx [flags] [path ...]
```

### Flags

`-f` - Force removal of files and directories.

`-i` - Prompt before removal.

`-r` - Recursively remove directories.

`-v` - Verbose output.
